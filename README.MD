## 介绍

虽然 Node.js 存在大量优秀的框架/库/工具，但它们都没有有效地解决主要问题 —— 项目架构与开发约束。

得益于 [typescript](https://www.typescriptlang.org/) 与 [Nest.js](https://github.com/nestjs/nest) 的出现，Node.js 开发者也可以感受如 `Spring` 那般威力强大的开发体验。

`Nest-Server` 基于 Nest.js 的功能封装，对国内 Node.js 开发者中常用的层级/模块/规范进行了约束，这些模块通过 [Nest-Server 配套工具](#工具与插件) 即可便捷地创建。
